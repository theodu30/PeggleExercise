# Peggle
This project is an attempt at recreating Peggle by PopCap Games.  
It was done in 3.5 hours as a class exercise.  

## Features
You can fire balls from a cannon at the top of the level.  
Pegs can be hit and reward you with points and disappear  
at the end of the turn.  
You have 10 balls. When you fire, you consume one. If it falls  
in the void, it is lost, if it falls in the bucket, you get one back.  
A bucket goes back and forth from left to right at the bottom.  
There are 25 orange pegs and 1 purple.  
The purple one changes places at the end of the turn if it was hit.  
If all balls are lost, you lose. If all orange pegs are hit, you win.  
You have a high score in the menu, and you can see your current score  
and number of balls left during the game.  

## Limitations
The purple peg should switch places every turn regardless  
of being hit or not.
